# 🚀 MiniTok AI SaaS

MiniTok AI adalah platform video short-style seperti TikTok yang menggabungkan:

- 🎬 YouTube Auto Trending Feed (API)
- 🛒 Affiliate Monetization System
- ❤️ Like + CTR Tracking
- 📊 Realtime Analytics (Firebase)
- 🔗 Smart Direct Link Redirect
- 📤 Social Share System
- 🧠 AI Simple Ranking (CTR + Views)

---

# ⚙️ Tech Stack

- HTML5 + CSS3 + JavaScript (Vanilla)
- Firebase Realtime Database
- YouTube Data API v3
- GitHub Pages Hosting
